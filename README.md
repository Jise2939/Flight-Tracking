## 專題概述

2026年2月28日，美以對伊朗發動大規模軍事打擊，伊朗隨即關閉領空並發射導彈反擊。國際民航組織（ICAO）發布最高級別警告，中東多國宣布禁止民航客機飛越衝突空域。

**24小時內，迪拜、多哈、阿布扎比三大樞紐航班取消率從不足1%飆升至35–52%，次日全面停擺。**

本專題通過航班數據可視化與行業分析，呈現戰爭對全球客運與貨運供應鏈的實質衝擊。
